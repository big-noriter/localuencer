# 로컬루언서 미나 - 기능 정리

## ✅ 완료된 기능

### UI/UX
  - 전체적인 웹사이트 디자인 및 레이아웃 (헤더, 푸터, 메인 콘텐츠 영역)
  - 반응형 디자인 (기본)
  - 라이트/다크 모드 테마 전환
  - 이미지 슬라이더 (홈페이지 히어로)
  - shadcn/ui 컴포넌트 (Button, Card, Input, Select, Tabs, Avatar, DropdownMenu 등) 활용

### 콘텐츠 페이지 (기본 구조 및 목업/부분 기능)
  - **홈페이지**: 미나 소개, 히어로 슬라이더, 주요 기능 카드, 콘텐츠 티저
  - **브이로그 목록 페이지**: 목업 데이터 기반 카드 UI
  - **쇼핑 목록 페이지**: 목업 데이터 기반 카드 UI
  - **가상여행 페이지**: Spatial.io 임베드 예시 (탭 UI)
  - **AI 가이드 페이지**: 기능 소개 UI (목업)
  - **사진엽서 페이지**: 사진 업로드, 관광지 선택, AI 생성 목업 UI
  - **Q&A 페이지**:
    - 질문 등록 (로그인 사용자)
    - Supabase DB 연동 (질문 저장)
    - OpenAI 연동 (AI 답변 자동 생성 및 저장)
    - Q&A 목록 표시 (DB 연동)
  - **SNS 피드 페이지 (신규)**:
    - SNS 통합 레이아웃
    - YouTube, Instagram (이미지, 릴스, 카드뉴스 탭), X(Twitter), Blogspot, News 각 채널별 목업 콘텐츠 페이지

### 사용자 인증 (Supabase Auth)
  - 회원가입 폼 및 기능
  - 로그인 폼 및 기능
  - 로그아웃 기능
  - AuthProvider 및 useAuth 훅
  - UserMenu (로그인 상태에 따른 UI 변경)
  - 사용자 프로필 DB 테이블 생성 및 가입 시 자동 생성

### 관리자 페이지 (기본 구조 및 목업/부분 기능)
  - 관리자 전용 레이아웃 (사이드바, 헤더)
  - 관리자 접근 권한 제어 (로그인 및 관리자 역할 확인)
  - **대시보드**: 목업 통계 UI
  - **페르소나 설정**:
    - UI (이름, 나이, 성격, 말투, 관심사, 금지어, 예시 문장 입력)
    - Supabase DB 연동 (API 통해 저장 및 조회)
  - **콘텐츠 생성기**: UI (콘텐츠 유형, 주제, 키워드 입력), 목업 콘텐츠 생성 로직
  - **콘텐츠 달력**: UI (월별 달력, 콘텐츠 목록), 목업 데이터 표시
  - **시스템 설정**: UI (API 키 입력, 알림 설정 등), 목업 데이터

### 백엔드 및 데이터베이스 (Supabase)
  - 데이터베이스 스키마 정의 (users, vlogs, qas, products, persona_settings 등)
  - RLS (Row Level Security) 기본 정책 설정
  - 초기 데이터 시딩 (관리자, 기본 페르소나, 샘플 콘텐츠)
  - Supabase 클라이언트 분리 (client.ts, server.ts)

### API
  - `/api/qna`: GET (Q&A 목록), POST (질문 등록 및 AI 답변 생성)
  - `/api/admin/persona`: GET (활성 페르소나 조회), POST (페르소나 저장)

## ⏳ 현재 작업 중 / 다음 작업 예정 기능

### 콘텐츠 기능 강화
  - 브이로그 상세 페이지 구현
  - 쇼핑 상품 상세 페이지 및 장바구니/주문 시스템 구현
  - 사진엽서 생성 기능 실제 AI 이미지 합성 연동 (Fal.ai 또는 유사 서비스)
  - AI 가이드 기능 구체화 (실시간 채팅, 구글맵/어스 연동)

### 관리자 기능 강화
  - Q&A 관리 페이지: 질문 목록, 답변 수정, 상태 변경 등
  - 쇼핑몰 관리 페이지: 상품 CRUD, 주문 관리
  - 분석 리포트 페이지: 사용자 활동, 콘텐츠 성과 시각화
  - 콘텐츠 달력 기능 DB 연동 및 예약 발행 기능
  - 시스템 설정 실제 환경 변수 연동 및 기능 활성화

### 사용자 경험 개선
  - 사용자 프로필 페이지 (정보 수정, 활동 내역)
  - 사용자 설정 페이지
  - 실시간 알림 시스템 (새 콘텐츠, Q&A 답변 등)
  - 소셜 로그인 (Google, Kakao 등)

### AI 기능 고도화
  - 콘텐츠 생성기 실제 OpenAI 연동 및 페르소나 기반 생성 로직 강화
  - Q&A 답변 품질 개선 (랭킹, 사용자 피드백 반영 등)

## 🔄 업데이트/개선 필요한 기능

  - **모든 목업 데이터 실제 DB 연동**: 브이로그, 쇼핑, AI 가이드, 사진엽서, SNS 피드 등
  - **관리자 페이지 기능**: 현재 UI만 있거나 목업 데이터로 동작하는 부분 실제 DB 연동 및 로직 구현
  - **반응형 디자인**: 모든 페이지 및 컴포넌트에 대한 세밀한 반응형 테스트 및 개선
  - **에러 핸들링 및 사용자 피드백**: API 요청 실패, 폼 유효성 검사 등 사용자에게 명확한 피드백 제공
  - **보안 강화**: RLS 정책 세분화, 입력값 검증 강화
  - **성능 최적화**: 이미지 최적화, 코드 스플리팅, API 응답 속도 개선
  - **테스트**: 단위 테스트, 통합 테스트, E2E 테스트 도입 고려
