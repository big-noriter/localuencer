# 로컬루언서 미나 (Localuencer Mina) - 제품 요구사항 문서 (PRD)

## 1. 개요

**프로젝트명**: 로컬루언서 미나 (Localuencer Mina)
**목표**: AI 기술을 활용하여 경주 지역에 특화된 매력적인 가상 인플루언서 '미나'를 제작하고, 사용자와 상호작용하며 경주 관광 정보 및 다양한 콘텐츠를 제공하는 웹 플랫폼 구축.
**타겟 사용자**:
  - 경주 여행을 계획 중이거나 관심 있는 국내외 관광객 (주로 10대 후반 ~ 30대)
  - AI 및 가상 인플루언서 기술에 관심 있는 사용자
  - 새로운 방식의 지역 관광 정보 습득을 원하는 사용자

## 2. 핵심 기능 및 요구사항

### 2.1. AI 인플루언서 '미나'
  - **페르소나**: 경주를 사랑하는 밝고 친근한 20대 초반의 여성 AI. Z세대 트렌드에 민감하며, 사용자에게 유용한 정보와 즐거움을 제공.
  - **시각적 표현**: 다양한 상황(캐주얼, 활동적, 클로즈업 등)에 맞는 고품질 2D/3D 이미지 및 영상(향후) 제공.
  - **상호작용**: Q&A, AI 가이드 등을 통해 사용자와 자연스러운 대화 및 정보 제공.

### 2.2. 사용자 대상 기능
  - **메인 페이지 (소개)**
    - 미나 소개 및 프로젝트 개요.
    - 주요 콘텐츠(브이로그, 가상여행 등) 티저 및 바로가기.
    - 히어로 이미지 슬라이더 (미나의 다양한 모습).
  - **브이로그 (/vlogs)**
    - 미나의 경주 일상 및 여행기 영상/글 목록 제공.
    - 각 브이로그 상세 페이지 (구현 예정).
  - **가상여행 (/virtual-travel)**
    - Spatial.io 등 메타버스 플랫폼 연동 (현재, 100년 전, 2000년 전 경주).
    - 각 시대별 경주 공간 임베드 및 설명.
  - **AI 가이드 (/ai-guide)**
    - 실시간 AI 가이드 세션 (음성/화상 채팅 - 구현 예정).
    - 맞춤 여행 일정 플래너 (구글맵/구글어스 연동 - 구현 예정).
    - 여행 편의 서비스 (할인 쿠폰, 관광 상품 예약 등 - 구현 예정).
  - **사진엽서 (/photo-postcard)**
    - 사용자 사진 업로드 기능.
    - 경주 관광지 배경 선택 기능.
    - AI 기반 사진 합성 및 엽서/4컷 사진 생성 (현재 목업, AI 연동 예정).
  - **Q&A (/qa)**
    - 사용자 질문 등록 기능 (로그인 필요).
    - AI 기반 답변 생성 및 표시 (OpenAI 연동).
    - 기존 Q&A 목록 조회.
  - **쇼핑 (/shop)**
    - 미나 추천 상품 목록 (경주 기념품, 패션, 뷰티 등).
    - 상품 상세 페이지 및 장바구니/주문 기능 (구현 예정).
  - **SNS 피드 (/sns/*)**
    - YouTube, Instagram (이미지, 릴스, 카드뉴스), X(Twitter), 블로그, 뉴스 등 미나의 SNS 채널별 콘텐츠 통합 제공.
    - 각 SNS 플랫폼 특성에 맞는 UI/UX.
  - **사용자 인증 (/auth/*)**
    - Supabase Auth 기반 회원가입 및 로그인.
    - 사용자 프로필 페이지 (구현 예정).
    - 사용자 설정 페이지 (구현 예정).
  - **공통 UI/UX**
    - 반응형 웹 디자인.
    - 라이트/다크 모드 테마 지원.
    - 직관적인 네비게이션 (헤더, 푸터).

### 2.3. 관리자 기능 (/admin/*)
  - **관리자 인증**: 관리자 계정만 접근 가능.
  - **대시보드**: 주요 활동 지표 시각화.
  - **페르소나 설정**: 미나의 AI 캐릭터 설정 (이름, 성격, 말투, 관심사, 금지어, 예시 문장 등). Supabase DB 연동.
  - **콘텐츠 생성기**: AI 기반 콘텐츠 초안 생성 (브이로그 스크립트, Q&A 답변, 상품 설명, SNS 포스트). OpenAI 연동.
  - **콘텐츠 달력**: 콘텐츠 발행 일정 계획 및 관리.
  - **Q&A 관리**: 사용자 질문 검토, 답변 수정/관리 (구현 예정).
  - **쇼핑몰 관리**: 상품 등록/수정/삭제, 주문 관리 (구현 예정).
  - **분석 리포트**: 사용자 활동 및 콘텐츠 성과 분석 (구현 예정).
  - **시스템 설정**: API 키(OpenAI 등) 관리, 시스템 알림 및 자동화 설정.

### 2.4. 기술 요구사항
  - **프론트엔드**: Next.js (App Router), React, TypeScript, Tailwind CSS, shadcn/ui.
  - **백엔드/DB**: Supabase (PostgreSQL, Auth, Storage).
  - **AI 연동**: Vercel AI SDK, OpenAI API (GPT 모델).
  - **배포**: Vercel.
  - **기타**: Embla Carousel (이미지 슬라이더), Spatial.io (가상여행).

## 3. 성공 지표 (예시)
  - 월간 활성 사용자 수 (MAU).
  - 페이지뷰, 평균 세션 시간.
  - Q&A 질문 수 및 사용자 만족도.
  - SNS 채널 팔로워 및 참여율 증가.
  - 쇼핑몰 상품 판매량 (향후).

## 4. 릴리즈 계획 (개략적)
  - **Phase 1 (현재)**: 기본 UI/UX, 주요 콘텐츠 페이지 (브이로그, 가상여행, AI가이드, 사진엽서, Q&A, 쇼핑 - 목업), 사용자 인증, 관리자 기본 기능(대시보드, 페르소나, 콘텐츠 생성기, 달력, 설정 - 목업/부분 DB 연동), SNS 피드 페이지.
  - **Phase 2**: Q&A AI 답변 고도화, 사진엽서 AI 연동, 관리자 기능 DB 연동 강화, 브이로그/쇼핑 상세 기능 구현.
  - **Phase 3**: AI 가이드 실시간 채팅 연동, 가상여행 고도화, 사용자 프로필/설정, 분석 리포트.
